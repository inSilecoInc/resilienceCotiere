3  Présentation des sources de données

3.1 Inventaire annuel de la Pointe à John

Les données proviennent des résultats d’inventaires annuels de poissons et de macroinvertébrés benthiques réalisés autour de la Pointe à John, près de Bergeronnes, au Québec, depuis 2012. Ces inventaires, supervisés par le Cégep de Sherbrooke, sont effectués par des étudiants inscrits au programme de Techniques de bioécologie. Le principal objectif est de former les étudiants aux pratiques d’échantillonnage en milieu marin. Les résultats incluent des données sur les événements d’échantillonnage, des observations biologiques, des mesures physico-chimiques, et le recouvrement des espèces (Ouellette and Sherbrooke, 2023).

  • Source : Inventaire Pointe à John
  • Accessibilité : Libre, Creative Commons Attribution 4.0 (CC-BY-4.0)
  • Type de données : Fichiers CSV
  • Couverture géographique : Pointe à John, Québec, Canada (latitude : 48.22 à 48.24, longitude : -69.57 à -69.53)
  • Traitement script : prc_inventaire_pointe_john
  • Fichiers de sortie :
    • events_pointe_john.csv
    • occurrences_pointe_john.csv
    • physico_chem_pointe_john.csv
    • species_recovery_pointe_john.csv

3.1.1 Étapes de traitement des données

Les données sont déjà disponibles dans un format standardisé; un effort minimal de traitement de données est ainsi nécessaire. Nous avons uniquement standardisés les noms de colonnes en utilisant janitor::clean_names() pour garantir des noms uniformes à travers le projet.

3.1.2 Structure des données traitées

Les tables produites par le processus de traitement sont décrites ci-dessous.

3.1.2.1 Table des événements d’échantillonnage

Variable Description
dataset_id Identifiant du jeu de données
event_id Identifiant unique de l’événement
country_code Code du pays (ISO 3166-1 alpha-2)
location Localisation de l’échantillonnage
decimal_longitude Longitude décimale
decimal_latitude Latitude décimale
year Année de l’échantillonnage
sampling_protocol Protocole d’échantillonnage
habitat Habitat échantillonné

3.1.2.2 Table des occurrences biologiques

Variable Description
event_id Identifiant de l’événement
occurrence_id Identifiant unique de l’occurrence
scientific_name Nom scientifique de l’organisme
vernacular_name Nom vernaculaire de l’organisme
occurrence_status Présence ou absence de l’organisme

3.1.2.3 Table des mesures physico-chimiques

Variable Description
event_id Identifiant de l’événement
measurement_type Type de mesure (ex. salinité, température)
measurement_value Valeur mesurée
measurement_unit Unité de mesure

3.1.2.4 Table des recouvrements d’espèces

Variable Description
event_id Identifiant de l’événement
measurement_type Type de recouvrement
measurement_value Pourcentage de recouvrement
